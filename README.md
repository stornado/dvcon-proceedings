# DVCon Proceedings

Download DVCon Papers, Posters, Presentations and Video Archive from https://dvcon-proceedings.org/

## Usage

```bash
git clone --depth 1 https://github.com/stornado/dvcon-proceedings.git
cd dvcon-proceedings
go mod tidy
go run main.go
```
